# Repository Rename Instructions

## New Repository Name: `azure-budget-static-site`

This name better reflects the cost-optimized nature of the project.

## Steps to Rename on GitHub:

1. Go to your repository: https://github.com/Tyra-Nyambura/azure-static-website-hosting
2. Click "Settings" tab
3. Scroll down to "Repository name"
4. Change to: `azure-budget-static-site`
5. Click "Rename"

## After Renaming:

1. Update any local remotes:
   ```bash
   git remote set-url origin https://github.com/Tyra-Nyambura/azure-budget-static-site.git
   ```

2. Update any documentation links that reference the old name
3. Inform collaborators about the name change

## Why This Name?

- **Budget**: Emphasizes cost optimization
- **Static Site**: Clear about the technology
- **Concise**: Easier to remember and type
- **Descriptive**: Immediately conveys the value proposition
